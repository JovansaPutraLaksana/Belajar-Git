# Belajar Git

Repository ini adalah dokumentasi pribadi untuk **mempelajari dan memahami Git sebagai sistem version control**.
Git merupakan alat penting yang membantu developer mengelola versi kode secara efisien dan terpadu dengan remote seperti GitHub.

---

## Tentang Repository

Di dalam repository ini, kamu akan menemukan catatan dan panduan dasar tentang Git, termasuk:

-  Konfigurasi awal Git
-  Dasar-dasar penggunaan Git (commit, add, status, dll)
-  Branch dan cara kerjanya
-  Remote repository dan cara sinkronisasi dengan GitHub

---

##  Struktur Folder dan File

Berikut adalah daftar file utama yang ada di repo ini:

| Nama File | Deskripsi Singkat |
|-----------|-------------------|
| `0-GitConfig.txt`  | Konfigurasi Git awal (username, email, dll) |
| `1-GitDasar.txt`   | Catatan dasar perintah Git |
| `2-GitBranch.txt`  | Penjelasan tentang branch di Git |
| `3-GitRemote.txt`  | Cara menghubungkan repositori lokal dengan GitHub |
---

## Cara Menggunakan

1. **Clone repository ini:**
   ```bash
   git clone https://github.com/JovansaPutraLaksana/Belajar-Git.git
   
2. **Baca setiap file .txt untuk memahami konsep dasar Git**
   
    Setiap file memberikan insight langkah demi langkah tentang topik tertentu.

3. **Praktikkan setiap perintah pada terminal/bash**
   
    Contoh: git init, git status, git add, git commit, dll untuk memperkuat pemahaman.
